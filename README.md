# ❤️ Music Heart Animation

A romantic heart animation website built with HTML, CSS, and JavaScript. The project features multiple glowing hearts floating across a dark background while background music plays, creating a beautiful and emotional visual experience.

## ✨ Features

- ❤️ Multiple Animated Hearts
- 🌙 Dark Aesthetic Background
- 🎵 Background Music Support
- ✨ Smooth Floating Animations
- 📱 Responsive Design
- ⚡ Lightweight & Fast

## 📸 Preview

A dark-themed romantic scene filled with glowing hearts moving smoothly across the screen while music plays in the background.

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/yourusername/music-heart-animation.git
```

### Run the Project

Simply open:

```text
index.html
```

in your browser.

You can also use VS Code Live Server for a better development experience.
